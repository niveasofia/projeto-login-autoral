## 📋 Descrição
&emsp;&emsp; Este projeto consiste no desenvolvimento de uma tela de login construída do zero, sem o uso de frameworks, com foco no aprendizado de conceitos básicos de HTML e breve exploração das possibilidades com CSS. <br>
&emsp;&emsp; O projeto foi desenvolvido como atividade prática na Unicesumar, sob orientação do Professor [Leonardo Rocha](https://github.com/leonardossrocha). No momento o projeto apresenta apenas a interface gráfica, sem qualquer funcionalidade implementada.

## 🚀 Como Executar
Utilizando o git clone, clone o repositório para seu dispositivo local e abra o index.html <br>

⚠️ Necessário ter o Git já devidamente instalado, e configurado em seu computador. <br>

Para clonar o repositório, acesse a pasta do seu computador através do terminal (VSCode, CMD). <br>
Utilize: cd + (endereço da pasta). Exemplo: cd C:\Users\usuário\documentos\projetos <br>
Utilize: git clone + repositório. Exemplo: git clone https://github.com/niveasofia/projeto-login-autoral.git <br>
Após a clonagem do repositório, localize a pasta onde ele foi clonado. <br>
O git clone baixa o repositório em seu computador, como uma pasta. <br>
Abra a pasta clonada, no explorador de arquivos do seu computador. <br>
Abra o arquivo index.html no navegador. <br>
